# 澳門公共停車場實時資訊 — 設計文檔

**日期**：2026-04-06
**狀態**：待審批

---

## 概述

構建一個簡單、乾淨的網頁應用，顯示澳門公共停車場的實時剩餘車位資訊，按用戶距離排序。替代 DSAT 官方頁面的複雜體驗。

---

## 系統架構

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────────┐
│   瀏覽器前端     │────▶│  Flask 後端代理   │────▶│  DSAT XML API       │
│   (HTML/CSS/JS) │◀────│  (app.py)        │◀────│  (dsat.apigateway.  │
│                 │     │                  │     │   data.gov.mo)      │
└─────────────────┘     └──────────────────┘     └─────────────────────┘
       │                        │
       │  獲取位置               │  解析 XML
       ▼                        ▼
  瀏覽器定位 API           計算距離 + 排序
```

### 技術棧

| 層級 | 技術 | 說明 |
|------|------|------|
| 前端 | 單一 HTML 文件 | 原生 HTML/CSS/JavaScript，無需構建工具 |
| 後端 | Python Flask | 輕量級 API 代理，解析 XML 並返回 JSON |
| 數據源 | DSAT XML API | `https://dsat.apigateway.data.gov.mo/car_park_maintance` |

---

## 數據流

1. 用戶打開頁面，瀏覽器請求定位權限
2. 前端獲取用戶經緯度後，調用後端 `/api/carparks` 接口
3. 後端向 DSAT API 發送請求，攜帶 APPCODE 授權
4. 後端解析返回的 XML，提取每個停車場的名稱、車位數據
5. 後端計算每個停車場與用戶的距離（Haversine 公式）
6. 後端按距離排序，返回 JSON 給前端
7. 前端渲染卡片列表
8. 每 30 秒自動刷新一次數據
9. 用戶可點擊刷新按鈕手動更新

## 後端 API 設計

### `GET /api/carparks?lat={lat}&lng={lng}`

**請求參數**：
- `lat`：用戶緯度（必需）
- `lng`：用戶經度（必需）

**返回格式**：
```json
{
  "updated_at": "2026-04-06T10:30:00+08:00",
  "user_lat": 22.1987,
  "user_lng": 113.5439,
  "carparks": [
    {
      "name": "栢力停車場",
      "distance_km": 0.3,
      "light_vehicle": {"available": 280, "total": 417},
      "heavy_vehicle": {"available": 0, "total": 0},
      "motorcycle": {"available": 30, "total": 32}
    }
  ]
}
```

**DSAT API 授權**：
- URL：`https://dsat.apigateway.data.gov.mo/car_park_maintance`
- Header：`Authorization: APPCODE 09d43a591fba407fb862412970667de4`

## 前端設計

### 佈局

- 頂部標題欄：「🅿️ 澳門停車場」+ 更新狀態 + 刷新按鈕
- 位置狀態提示：顯示是否已獲取用戶位置
- 停車場卡片列表：按距離從近到遠排列

### 卡片設計

每張卡片包含：
- 停車場名稱（左側，粗體 16px）
- 距離（右側，14px）
- 三類車位數據：輕型汽車 🚗、重型汽車 🚛、電單車 🛵

### 車位充足度顏色規則

| 顏色 | 條件 | 說明 |
|------|------|------|
| 🟢 綠色 (#4caf50) | 剩餘 > 50% | 車位充足 |
| 🟡 橙色 (#ff9800) | 剩餘 20%-50% | 車位一般 |
| 🔴 紅色 (#f44336) | 剩餘 < 20% | 車位緊張 |

左側色條（4px）根據輕型汽車剩餘比例顯示對應顏色。

### 更新機制

- 自動刷新：每 30 秒調用一次 API
- 手動刷新：點擊 🔄 按鈕立即更新
- 更新時顯示載入狀態

## 錯誤處理

| 場景 | 處理方式 |
|------|----------|
| 用戶拒絕定位 | 顯示提示，要求授權位置權限 |
| DSAT API 超時 | 顯示「數據獲取失敗，請稍後重試」 |
| 網絡斷開 | 顯示「網絡連接異常」 |
| XML 解析錯誤 | 記錄錯誤，顯示友好提示 |
